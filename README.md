# Projeto de Cardápio Digital com Spring Boot e PostgreSQL
Este é um projeto que utiliza o framework Spring Boot e o banco de dados PostgreSQL para criar um cardápio digital. O sistema permite que os usuários insiram informações sobre pratos, incluindo nome, preço e foto, e essas informações são armazenadas no banco de dados.

## Funcionalidades
O sistema de cardápio digital possui as seguintes funcionalidades:

Adição de pratos: Os usuários podem adicionar informações sobre os pratos, incluindo nome, preço e foto.
Visualização do cardápio: Os usuários podem visualizar todos os pratos existentes no cardápio.
Atualização de pratos: Os usuários podem editar as informações de um prato existente, incluindo nome, preço e foto.
Remoção de pratos: Os usuários podem excluir pratos do cardápio.
## Tecnologias Utilizadas
O projeto é desenvolvido utilizando as seguintes tecnologias:

Spring Boot: Um framework Java que simplifica o desenvolvimento de aplicativos baseados em Spring. Ele fornece uma configuração inicial e um conjunto de bibliotecas para facilitar o desenvolvimento.
PostgreSQL: Um banco de dados relacional de código aberto que oferece uma solução escalável e robusta para armazenar dados.
## Configuração do Ambiente
Para configurar o ambiente de desenvolvimento, siga as etapas abaixo:

Certifique-se de ter o Java Development Kit (JDK) instalado na versão adequada.
Baixe e instale o PostgreSQL no seu sistema operacional.
Configure as variáveis de ambiente do Java e do PostgreSQL, se necessário.
Baixe o projeto do repositório de código-fonte.
## Executando o Projeto
Para executar o projeto, siga os passos a seguir:

Abra o projeto em sua IDE preferida.
Configure as informações de conexão com o banco de dados PostgreSQL no arquivo de configuração do Spring Boot.
Execute o projeto a partir da classe principal (geralmente chamada de Application ou Main).
O projeto será iniciado e estará disponível em http://localhost:8080.
